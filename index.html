1. Income — collect Cr + Sponsor bonuses

3. Market — buy upgrades, hire staff, Sponsor
4. Workshop — install, repair, search deck
5. Race — Overheat → Seg 1→2→3
6. End Phase — CP, Fatigue, advance
Reckless Driving
Declare before each Segment (simultaneous)
Benefit: +2 Segment Score
Risk: D6 — 1 or 2 = −1 Reliability
+1 Reckless Counter (never resets)
Counter ≥5 → Fatigue Token
Each token: −1/Seg this Race
Overheat Check
Check: Heat > Cooling?
Step 1: −1 Reliability (always)
Step 2: Roll D6
Roll ≥ Overtemp: safe
Roll < Overtemp: −2 more R this Race
R ≤ 0 = DNF (−1 CP, −1 Cr)
