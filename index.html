Rules
1 — Players
2 — Race Calendar (revealed BEFORE auction)
Study the calendar before bidding on cars. A Technical-heavy calendar favours the MX-5 and Audi TT. High-Speed favours the Mustang and 370Z.
3 — Car Auction
How the auction works:
• Cars are revealed one at a time in random order
• Every player bids from their starting 12 Credits
• Opening bid: 1 Cr. Any player may bid. Raise or pass.
• Highest bidder wins the car and pays from their 12 Cr
• Whatever you spend on your car comes off your Race 1 budget
• If nobody bids, the car is unclaimed (no one gets it)
4 — Start Championship
Complete the auction to start.

Round Sequence
1. Income — collect Cr (last place first in market)
2. Fuel Station — optional swap
3. Market Phase — buy upgrades, hire staff, auction
4. Workshop Phase — spend WU to install / repair / search deck
5. Race — Overheat checks → Segments 1–3
6. End Phase — CP, Fatigue decay, advance calendar
Segment Scoring
Score = (Your Stat A − Demand A)
+ (Your Stat B − Demand B)
+ Reckless Bonus (+2 if declared)
− Fatigue Penalty (−1 per Token)
± Weather / Hazard effects
Scores are often negative early in the season — that's normal. Only the DIFFERENCE between players decides the winner.
Reckless Driving
Declare before each Segment (simultaneously)
Benefit: +2 to Segment Score
Risk: Roll D6 — on 1 or 2, lose 1 Reliability
+1 to Reckless Counter (never resets)
Counter hits 5 → 1 Fatigue Token
Each Fatigue Token → −1 Score/Seg this Race
Engineer — Deck Search
1. Pick up your face-down deck
2. Browse it freely — no rush, no announcement
3. Choose any card (or take nothing — free action)
4. Pay its Buy cost to the bank
5. Card goes to your Garage
6. Shuffle the deck

GitHub Pages Fix
If you're getting a GitHub Pages / Jekyll error:
1. In your repo, create a file called .nojekyll (empty file, that exact name with the dot)
2. Make sure index.html is in the ROOT of your repo — not inside a /docs folder
3. In Settings → Pages → Source, choose "Deploy from a branch" → branch: main → folder: / (root)
4. The Jekyll error happens because GitHub tries to build the site as a Jekyll project. The .nojekyll file stops that.
